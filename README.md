This repository contains  R scripts to accompany a comment article on Muda and Yangxuan (2026), "Test Statistics for Confirmatory Factor Analysis Under Non-Normality  and Large $p/n$ Ratios: A Comment on Muda and Yangxuan (2026)", by Foldnes and Grønneberg, in the SEM journal 2026. 
One script is the original simulation script used by  Muda and Yangxuan (2026). The other script contains correct implementation of benchmark stastistics, as well as some additional simulation conditions. Also, a script for transforming p-values from a Foldnes, Grønneberg and Moss study in Behavior Research Methods (2026) in high dimensions are provided.
